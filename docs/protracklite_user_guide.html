ProtrackLite User Guide

Application: ProtrackLite

Main Login URL: https://tasks.omnihire.in/solulever/login

Audience: Team members using the application for task planning, weekly review, and personal reporting

Prepared On: 13 April 2026

1. What This App Is Used For

ProtrackLite is used to manage personal work in a structured way. It helps each user log in, create and maintain tasks, plan work for the week, review Monday updates, and track effort through personal reports.

The most important pages for day-to-day use are:

  • Login for secure access and password reset
  • Create Task for adding one task at a time or multiple tasks with AI
  • Monday Report for weekly planning and weekly review
  • My Reports for effort, performance, and workload visibility

2. Logging In

2.1 Open the Login Page

Open the following page in your browser:

https://tasks.omnihire.in/solulever/login

This page is specific to the solulever organization, so always use the correct org login URL.

2.2 Enter Your Credentials

  1. Enter your company email address.
  2. Enter your password.
  3. Click Login.

After successful login, you will be taken into the main workspace.

2.3 If You Forgot Your Password

The recommended way to access the app is to use the built-in password reset flow.

  1. Open the login page.
  2. Click Forgot Password?
  3. In the popup window, enter your company email address.
  4. Solve the simple captcha shown in the popup.
  5. Click Send Reset Password.

The system sends a temporary password to your email address. Use that temporary password to log in.

Important: Temporary passwords are time-sensitive. If the reset message is not available or expires, repeat the forgot-password process.

3. Creating a Task

3.1 Open the Create Task Page

After login, go to Create Task from the left navigation, or open:

https://tasks.omnihire.in/solulever/tasks/new

3.2 Create One Task Manually

The standard task form is designed for fast entry. Fill the fields as follows:

Field What To Do
Add as Backlog item Check this if the task is only for future reminder and you do not want to schedule it right now.
Project Select the relevant project from the dropdown.
Name Enter a short, clear task title.
Category Select the high-level task category first.
Activity Type Select the detailed activity under the chosen category.
Status Select the current state if the task is not backlog.
Start Date / End Date Set planned dates for execution.
Estimated Hours Enter expected effort.
Private task Use this when the task should stay private.
Description Add supporting notes or context if needed.

3.3 Creating a Backlog Task

A backlog task is a task that you want to remember, but do not want to prioritize or schedule yet.

  1. Check Add as Backlog item first.
  2. Complete only the basic fields such as project, name, category, activity type, and optional description.
  3. Save the task.

When backlog is enabled, schedule-related fields are hidden because backlog items do not need start date, end date, or effort at the time of creation.

3.4 Save and Continue

When you click Save Task, the app stays on the same page so you can continue adding more tasks. The newest task appears at the top of the My Tasks list on the same screen.

Good practice: Use clear task names. A good task title makes reporting, searching, and weekly review much easier.

4. Creating Multiple Tasks With AI

4.1 When To Use This

Use Add Multiple Task when you have several tasks in your mind and do not want to create them one by one.

4.2 How It Works

  1. Open the Create Task page.
  2. Click Add Multiple Task.
  3. A popup window opens.
  4. Enter one task per line in free-flow language.
  5. Optionally check Private tasks if all items should be private.
  6. Click Create Tasks With AI.

4.3 What AI Extracts

The AI parser attempts to identify the following details from each line:

  • task title
  • possible start date
  • possible end date
  • possible effort

Each parsed task is then saved using your default project and default activity type settings.

4.4 Example Input

You can write tasks like this:

  • Prepare client deck by Thursday
  • Review infra access request tomorrow
  • Research hiring dashboard options next week
  • Document sprint plan by 18 April 4h

4.5 After AI Task Creation

After processing, the app returns you to the Create Task page and shows a confirmation with the task codes created. You can then:

  • continue adding more tasks
  • review the newly created tasks from the task list on the same page
  • open any task later to refine dates, effort, status, or notes

Important: AI parsing is a helper, not a replacement for review. If a task needs precise dates or effort, open the task afterward and confirm the details.

5. Using the Monday Report

5.1 When To Use It

The Monday Report should be used every Monday as part of your weekly planning and presentation process.

Open:

https://tasks.omnihire.in/solulever/reports/monday

5.2 What The Monday Report Shows

The page is structured into clear sections:

  • Last Week Closed: number of tasks closed last week
  • Last Week Effort: total hours booked last week
  • Booked %: booked effort versus available weekly hours
  • This Week Tasks: tasks starting in the current week
  • Pending Tasks: active open tasks from before this week that are not stalled
  • Completed Tasks: items closed last week, with planned versus actual effort
  • Stalled Tasks: tasks currently marked stalled

5.3 How To Read It In A Weekly Meeting

Use the Monday Report as your weekly presentation page.

  1. Start with the top summary cards to explain last week’s closure and effort.
  2. Review This Week Tasks to explain what you plan to start this week.
  3. Review Pending Tasks to call out work still open and any overdue items.
  4. Use Completed Tasks to explain what was closed and whether actual effort matched plan.
  5. Review Stalled Tasks if something is blocked.

The page already highlights logged hours and deadline state so the important numbers are easy to present on screen.

5.4 Add Tasks From Backlog Into This Week

If you want to pull work into the current week directly from backlog:

  1. Open the Monday Report.
  2. In the This Week Tasks section, click Add From Backlog.
  3. Select the backlog tasks you want to plan for this week.
  4. Adjust this week’s end date if required.
  5. Adjust planned effort if required.
  6. Click Add To This Week.

The system moves the selected tasks into the week plan with a start date for the current week and the planned values you choose in the popup.

6. Using My Reports

6.1 Open My Reports

Open:

https://tasks.omnihire.in/solulever/reports/overview

6.2 What This Page Is For

My Reports gives you a personal performance and effort view. It is useful for self-review, planning, and management visibility.

6.3 Main Sections On The Page

Section Meaning
Week Wise Effort Summary Booked hours against available hours for the last four weeks.
This Month Total booked effort, available hours, booked percentage, and chargeable effort for the current month.
Top 5 Tasks This Month The tasks where you booked the highest effort this month.
Pending Task List Delay distribution of current pending tasks, shown as a chart.
Burn Down Weekly comparison of assigned versus closed tasks for the last five weeks.
Task Type Wise Allocation How your monthly effort is distributed across activity types.

6.4 How To Use My Reports Properly

  • Check whether your booked hours are close to your available hours.
  • Look at the top tasks to understand where most time went.
  • Monitor pending-task delay to catch slippage early.
  • Review the burn-down trend to see whether closure is keeping pace with assignment.
  • Review activity allocation to understand how your time is split across task types.

7. Recommended Weekly Working Pattern

  1. Use Forgot Password if you cannot sign in.
  2. Create or refine your tasks before starting work.
  3. Use Add Multiple Task when you have many tasks to capture quickly.
  4. Keep backlog items for future work you do not want to schedule yet.
  5. Every Monday, open Monday Report and align your week.
  6. During the week, keep tasks updated so Monday reporting stays accurate.
  7. Use My Reports to monitor effort and closure trends.
